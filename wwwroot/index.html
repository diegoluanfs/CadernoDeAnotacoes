Nível 1: Fundamentos

HTML, CSS, e JavaScript:

  • Estrutura básica de uma página web: Aprenda a criar a estrutura fundamental de uma página HTML, incluindo tags como <html>, <head>, <body>, etc.

    • Exemplo: Uma estrutura HTML básica seria assim:

    
            <!DOCTYPE html>
            <html lang="en">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>Minha Página</title>
            </head>
            <body>
                <!-- Conteúdo da página aqui -->
            </body>
            </html>

    Existem diversas outras tags como por exemplo:

    Estrutura Básica:

    • <!DOCTYPE html>: Define a versão do HTML.
    • <html>: Elemento raiz da página HTML.
    • <head>: Contém metadados da página.
    • <title>: Define o título da página exibido na barra de título do navegador.
    • <body>: Contém o conteúdo da página.

    Texto:

    • <h1> a <h6>: Títulos de diferentes níveis.
    • <p>: Parágrafo.
    • <br>: Quebra de linha.
    • <hr>: Linha horizontal.

    Links e URLs:

    • <a>: Hiperlink.
    • <link>: Vincula recursos externos, como folhas de estilo.

    Listas:

    • <ul>: Lista não ordenada.
    • <ol>: Lista ordenada.
    • <li>: Item da lista.

    Imagens e Multimídia:

    • <img>: Inclui uma imagem.
    • <audio>: Elemento para áudio.
    • <video>: Elemento para vídeo.

    Formulários:

    • <form>: Contêiner para formulários.
    • <input>: Campo de entrada.
    • <textarea>: Área de texto.
    • <button>: Botão.
    • <select>: Lista suspensa.
    • <option>: Opção em uma lista suspensa.
    • <label>: Rótulo para elementos de formulário.

    Tabelas:

    • <table>: Tabela.
    • <tr>: Linha da tabela.
    • <th>: Cabeçalho da tabela.
    • <td>: Célula da tabela.

    Estilo e Layout:

    • <div>: Bloco de divisão (usado para agrupar elementos e aplicar estilos).
    • <span>: Span (usado para aplicar estilos a partes específicas do texto).

    Meta Informações:

    • <meta>: Fornece informações sobre a página, como conjunto de caracteres, autor, descrição, etc.

    Semântica:

    • <header>: Define um cabeçalho para a seção ou página.
    • <footer>: Define um rodapé para a seção ou página.
    • <nav>: Define uma barra de navegação.
    • <article>: Define um conteúdo independente e autossuficiente.
    • <section>: Define uma seção em um documento.
    • <aside>: Define conteúdo relacionado, muitas vezes ao lado do conteúdo principal.

    Formatação de Texto:

    • <strong>: Texto forte (importante).
    • <em>: Texto enfatizado (itálico).
    • <u>: Texto sublinhado.
    • <s>: Texto riscado.
  • Estilização com CSS: Explore as propriedades de estilo do CSS para dar design e layout à sua página. Compreenda seletores, propriedades e valores.

    • Exemplo: Vamos estilizar um elemento <div> com uma cor de fundo e texto centralizado:

    
            div {
                background-color: #3498db;
                color: #fff;
                text-align: center;
                padding: 20px;
            }

    Temos também elementos css

    Seletores:

    • Seletores de Tipo
    • Seletores de Classe
    • Seletores de ID
    • Seletores de Atributo
    • Seletores Combinados

    Propriedades:

    • color: Cor do texto.
    • font-family: Família da fonte.
    • font-size: Tamanho da fonte.
    • font-weight: Peso da fonte.
    • text-align: Alinhamento do texto.
    • margin: Margens externas.
    • padding: Preenchimento interno.
    • background-color: Cor de fundo.
    • border: Borda.

    Layout:

    • display: Tipo de exibição (bloco, linha, etc.).
    • position: Posicionamento (relativo, absoluto, fixo).
    • float: Flutuação de elementos.
    • clear: Limpa o comportamento de flutuação.
    • flexbox: Layout flexível.
    • grid: Layout em grade.

    Transformações e Transições:

    • transform: Transformações (escala, rotação, etc.).
    • transition: Transições suaves entre estados.

    Responsividade:

    • media queries: Aplicação de estilos baseada em características do dispositivo.

    Animations:

    • @keyframes: Define animações.
    • animation: Aplica animações a elementos.

    Pseudoelementos e Pseudoclasses:

    • ::before: Pseudoelemento antes do conteúdo.
    • ::after: Pseudoelemento após o conteúdo.
    • :hover: Estado quando o mouse passa sobre o elemento.
    • :nth-child: Seletor de filho específico.

    Flexibilidade e Grid:

    • flex: Propriedades flexíveis.
    • grid-template: Template para layouts de grade.
  • Interação do usuário com JavaScript: Inicie no mundo da programação do lado do cliente. Aprenda a manipular o DOM, responder a eventos e criar interatividade.

    • Exemplo: Vamos criar um simples script que exibe uma mensagem quando um botão é clicado:

    
            <button id="meuBotao">Clique-me</button>
            <script>
                document.getElementById('meuBotao').addEventListener('click', function() {
                    alert('Você clicou no botão!');
                });
            </script>

Git e Controle de Versão:

  • Comandos básicos do Git: Familiarize-se com comandos essenciais como `init`, `add`, `commit`, `push` e `pull` para rastrear alterações e colaborar em projetos.

    • Exemplo: Um fluxo de trabalho comum pode ser:

    
            git init  # Inicializa um novo repositório Git
            git add .  # Adiciona todos os arquivos ao próximo commit
            git commit -m "Primeiro commit"  # Cria um commit com uma mensagem
            git push origin master  # Empurra as alterações para o repositório remoto
  • Trabalhar com repositórios remotos: Entenda como interagir com repositórios remotos em plataformas como GitHub, GitLab ou Bitbucket.

    • Exemplo: Clonar um repositório remoto para o seu computador:

    
            git clone https://github.com/nomeusuario/exemplo.git
